# INSTALAÇÃO 0.4.5 — Parakeet TAGARELA + Mem0

## 1. Dependências novas

pip install "onnx-asr[cpu,hub]" huggingface_hub mem0ai qdrant-client

Se quiser tentar GPU via ONNX Runtime, instale onnxruntime-gpu conforme sua versão de CUDA.
A configuração padrão usa provider "auto" para não quebrar instalação CPU.

## 2. Baixar o modelo Parakeet TAGARELA ONNX

Opção recomendada: baixar manualmente para a pasta esperada.

python -c "from huggingface_hub import snapshot_download; snapshot_download(repo_id='alefiury/parakeet-tdt-0.6b-v3-ptBR-TAGARELA-onnx', local_dir='models/parakeet-tdt-0.6b-v3-ptBR-TAGARELA-onnx')"

## 3. Ativar STT

No config.py:

LOAD_STT = True
STT_ENGINE = "parakeet"

Se quiser voltar para Whisper:

STT_ENGINE = "whisper"

## 4. Mem0

Antes de usar Mem0, garanta que o Ollama tem os modelos:

ollama pull qwen2.5:7b-instruct
ollama pull nomic-embed-text

O Mem0 usa:
- qwen2.5:7b-instruct como LLM
- nomic-embed-text como embedder
- Qdrant local em mem0_qdrant

## 5. Memória manual segura

Durante teste por texto ou voz:

/mem0 lembrar Natan prefere que respostas de código venham com arquivo e função exata.

Auto-save fica desligado por padrão:

MEM0_AUTO_SAVE_INTERACTIONS = False

Só ligue quando o STT estiver confiável.
