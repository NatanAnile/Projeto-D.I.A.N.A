# Projeto Diana 0.4.14 — INPUT TERMINAL FIX

Correção pequena de usabilidade do terminal.

## Corrigido

- O modo texto agora fica sempre ativo, mesmo com STT ligado.
- O terminal voltou a exibir prompt constante:
  - `Digite sua mensagem:`
- A fila de texto agora tem prioridade sobre o push-to-talk.
- O push-to-talk por tecla continua disponível quando não há mensagem digitada.

## Motivo

Na 0.4.13, quando `LOAD_STT=True`, a thread de input textual não era iniciada.
Na prática, o terminal mostrava que dava para digitar, mas o loop só processava voz via `F`, travando o uso por texto.
