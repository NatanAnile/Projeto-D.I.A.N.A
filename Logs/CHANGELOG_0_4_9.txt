# Diana 0.4.9 — STT externo + correções de memória/persona

- Modelo Parakeet/TAGARELA não baixa mais dentro da pasta do projeto por padrão.
- `PARAKEET_MODEL_PATH` agora usa cache externo: `~/.cache/diana/models/...`.
- Adicionado `DIANA_CACHE_DIR` para trocar a pasta de cache sem editar código.
- Adicionado `stt/stt_postprocessor.py` para correções pós-STT.
- Corrigidos casos como `taquinha` → `Diana`, `Allen` → `Alien`, `Esgate` → `O Resgate`.
- Adicionadas travas anti-flerte, anti-meiguice, anti-`Neitanzinho` e anti-ações físicas falsas.
- Mem0 auto-save agora usa memória direta determinística quando possível.
- Mem0 direto usa `infer=False` para evitar erro de parsing JSON na extração local.
- Parser de preferências não salva pergunta/continuação como parte do valor.
- Suporte a `filme_alien_favorito = Aliens: O Resgate`.
- QueryGate reconhece perguntas pessoais com `do pai`.
- Mantidos modelo auxiliar, Query Planner auxiliar e Resumidor auxiliar.
- Raiz continua limpa e testes locais fora do ZIP final.
